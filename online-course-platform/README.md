# 🎓 LearnHub: Online Course Platform API

This is a fully functional FastAPI backend system built for the **FastAPI Internship Training - Final Project**. It simulates a real-world online learning platform where students can browse courses, add them to a wishlist, and enroll.

## 🚀 Features Implemented
This project successfully implements all Day 1 – Day 6 core concepts:
* **RESTful GET & POST APIs** * **Pydantic Data Validation** (Request bodies, constraints, and custom error messages)
* **Helper Functions** (Modular business logic for fee calculations and filtering)
* **Full CRUD Operations** (Create, Read, Update, Delete with proper status codes)
* **Multi-step Workflows** (Wishlist system with bulk enrollment capabilities)
* **Advanced Queries** (Keyword search, multi-parameter filtering, sorting, and pagination)

## 🛠️ Tech Stack
* **Framework:** FastAPI
* **Validation:** Pydantic
* **Server:** Uvicorn
* **Language:** Python 3.x
